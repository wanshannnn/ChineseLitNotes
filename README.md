#### now
1. 目前这个网站依然是并不完善的半成品，先放上来以push自己。
2. 最初打算开始这个项目的想法，也即目前这个网站的主要功能，是作为一个纯粹的文档网站，为忙于期末复习的中文系本科学生（或考研党）提供有用且便携的复习笔记。
- 这也是基于我本科学习期间的困扰：我们本该做的事情是更多地阅读和学习、思考和交流，但在每学期结束的时候却总是会面临一场诡异的“背诵教材”任务，并需要将大量的时间用于整理笔记和刻板背诵。教材太过笨重，笔记太过混乱，这些麻烦令我们失去了太多宝贵的学习、睡眠与游戏时间。
3. 加入评论系统。每个文档的底部设有评论区，可以在这里围绕文档内容留言交流，也可以在这里补充知识性信息。
4. 加入网站浏览量记录板块。
