#### 简介
1. 目前这个网站依然是并不完善的半成品，先放上来 push 自己。
2. 最初打算开始这个网站的想法，也即目前这个网站的主要功能，是作为一个纯粹的文档网站，为忙于考试的中文系本科学生提供 easily available & portable 的复习笔记。
- （这也是我本科时候的一大困扰：我们本该做的事情是更多地读书、思考、交流，但在每学期结束的时候却总是会面临一场诡异的“背诵教材”任务，为之劳身劳心，却考后即忘。我们应该尽量减少用于应付考试的时间，去做我们该做的事，比如有益的学习，或者睡觉与游戏。）
